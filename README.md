📊 Simple Linear Regression from Scratch
🚀 Overview

This project implements Simple Linear Regression using pure mathematics, without relying on machine learning libraries like scikit-learn.

The objective is to understand how linear regression works internally by manually calculating the model parameters and visualizing the results.

🧠 Concept

Linear Regression finds the best-fit line for a dataset using the equation:

y=a+bx

Where:

a → Intercept
b → Slope
🔧 Features
✅ Manual calculation of slope (b) and intercept (a)
✅ Prediction of output values
✅ Visualization of:
Data points
Regression line
✅ Clean and simple implementation
📈 Final Output

Regression Equation:

y = 0.84 + 0.65x
